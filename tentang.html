Tentang Saya

Halo! Saya Aarif Rahmaan Jalaluddin Faqiih, atau biasa dipanggil Faqih. Saya lahir di Banyumas dan kini tinggal di Purwokerto, sedang menempuh studi Teknik Informatika di Telkom University Purwokerto. Ketertarikan saya pada dunia pengembangan web bermula sejak SMP ketika pertama kali mempelajari HTML, dan berkembang menjadi fokus utama pada frontend development. Saya juga memiliki minat besar pada produksi musik, terinspirasi oleh karya Alan Walker sejak saya SD.


Dalam diri saya, minat terhadap produksi musik elektronik merupakan ketertarikan yang pertama saya temukan sebelum pengembangan web. Minat ini terus menyelimuti saya dan menjadi bagian yang sangat penting dari kreativitas saya, di samping pengembangan web. Sebagai tambahan, saya memiliki kebiasaan untuk membuat catatan dari film yang saya tonton.


Magang PT. Braincode Solution

Pada bulan April 2024, PT. Braincode Solution membuka rekrutmen terbuka di SMK saya. Setelah lolos tahap wawancara, saya mulai menjalani magang di cabang Purwokerto pada 30 April hingga 31 November 2024. PT Braincode sendiri berpusat di Jakarta.


Selama magang saya berperan sebagai frontend developer, mengerjakan pembuatan dashboard admin yang mencakup berbagai jenis chart seperti pie chart, bar chart, dan lainnya, serta tabel dengan menggunakan library. Selain itu, saya juga terlibat dalam pembuatan UI design untuk dashboard dan ERP. Pengalaman ini memperkenalkan saya pada dunia kerja, seperti meeting dengan project manager dan mengerjakan tugas yang diberikan, diskusi dengan tim, bertanya kepada developer senior, dan momen-momen berkesan lainnya. Beberapa proyek saya tampilkan di halaman portofolio.

Magang PT. Mega Giga Solusindo

Saya mengikuti program magang yang diadakan sekolah saya selama 6 bulan sejak 3 April hingga 29 September 2023 di PT. Mega Giga Solusindo. Selama magang saya membuat website CRUD, melakukan user research untuk keperluan pengembangan website dengan metode survey online maupun wawancara secara langsung, dan membuat website company profile PT. Mega Giga Solusindo dengan framework Vue.js dan toko Azka Jaya Trophy tanpa framework. Selain itu, saya membuat flowchart, manual book, dan scenario test, serta berkesempatan menyimak interview calon programmer dengan HRD.
